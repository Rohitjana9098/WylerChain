# Technology Stack

## Core
- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Runtime**: Node.js

## Styling & UI
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Design System**: Stitch "WylerChain Premium Launch UI"

## Engineering Tools
- **Deployment**: Vercel (recommended)
- **State Management**: React Hooks (State/Context)
- **Data Fetching**: Fetch API (mocked)

## Stitch Integration
- **StitchMCP**: Used for design system application and screen generation.
