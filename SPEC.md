# WylerChain Project Specification

## Objective
Develop and launch the WylerChain ecosystem application, transitioning from a high-fidelity static landing page to a fully functional Next.js 14 Web3 dashboard.

## Technology Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS + Framer Motion
- **Icons**: Lucide React
- **Authentication**: Social Login (Google, Apple) + Email (Mocked -> Transitioning to Stitch)
- **Data Layer**: Stitch (Pending Integration)
- **Aesthetic**: "Obsidian Void" / Premium Dark Mode (Glassmorphism, Vibrant Accents)

## Core Features
1. **Landing Page**: Modular interactive hero with boneyard skeleton loading.
2. **Authentication**: Seedless social login interface.
3. **Dashboard Shell**: Unified hub for WylerOS modules.
4. **Modules**:
   - **Wallet**: Balance tracking, Send/Receive/Bridge/Stake (Real-time).
   - **Staking**: APY tracking and asset management.
   - **Creators**: Social interaction and tipping.
   - **NFTs**: Collection gallery and minting.
   - **Explore**: Ecosystem navigation.

## Status: IN PROGRESS
Current focus: Connecting Dashboard components to Stitch for live data.
